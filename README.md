1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Run the development server**

   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) in your browser.

   2. **test emails**

   If the email contains “admin” → the role will be Admin
   If the email contains “pm” → the role will be Project Manager
   Otherwise → the role will be Developer

   Example:

   admin@test.com → Admin
   pm123@example.com → Project Manager
   rami@example.com → Developer
